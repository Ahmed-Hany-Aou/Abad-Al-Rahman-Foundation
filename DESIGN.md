# DESIGN: Abad Al-Rahman Foundation

## Color Strategy
Restrained

## Core Palette (OKLCH)
- **Primary (Charity Green)**: `oklch(45% 0.12 150)` - Represents growth, life, and humanitarian aid.
- **Accent (Dignity Gold)**: `oklch(80% 0.1 85)` - Used for certificates, highlights, and trust indicators.
- **Neutral (Tinted Slate)**: `oklch(20% 0.005 150)` to `oklch(98% 0.005 150)`.

## Typography
- **LTR (English, etc.)**: Inter (Geometric, modern, professional).
- **RTL (Arabic)**: Cairo (High legibility, modern, and culturally aligned).

## Principles
1. **Mirrored Rhythm**: Spacing must be identical in RTL and LTR, achieved through CSS Logical Properties.
2. **Elevated Content**: Use subtle shadows (`oklch`) and whitespace to create hierarchy, avoiding nested cards.
3. **Motion with Purpose**: Gentle fade-ins and scale-ups for impact metrics to make the data feel "alive."
